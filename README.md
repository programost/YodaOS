# YodaOS 1.2

> Минималистичная 32-битная операционная система, написанная с нуля на C и ассемблере.  
> Вдохновлена эстетикой 90-х, создана для обучения и ностальгии.

![YodaOS Screenshot](screenshot.png)

## ✨ Возможности

- 🖥️ **Текстовый режим VGA** 80x25, 16 цветов, аппаратный курсор.
- ⌨️ **Клавиатура** с поддержкой Shift, Caps Lock и стрелок.
- 💾 **Собственная файловая система YFS** (раздел типа `0x7F`):
  - Создание, удаление, чтение и запись файлов и папок.
  - Сохранение данных на реальный ATA-диск.
- 🧰 **Драйвер ATA PIO** с автоматическим определением размера диска.
- 📅 **CMOS/RTC** – дата и время.
- 🧩 **Встроенный текстовый редактор** `ynan`.
- 🔄 **Полноценная перезагрузка** и выключение.

## 🧱 Команды оболочки

| Команда               | Описание                                 |
|----------------------|------------------------------------------|
| `help -p [page]`     | Список команд по страницам               |
| `sysinf`             | Информация о системе (RAM, диск)         |
| `clear`              | Очистка экрана                           |
| `reboot` / `shutdown`| Перезагрузка / выключение                |
| `format`             | Создать раздел YFS на всём диске         |
| `ls` / `pwd` / `cd`  | Навигация по файловой системе            |
| `mkdir <dir>`        | Создать папку                            |
| `touch <file>`       | Создать пустой файл                      |
| `cat <file>`         | Вывести содержимое файла                 |
| `ynan <file>`        | Текстовый редактор (ESC для сохранения)  |
| `rm -f <file>`       | Удалить файл                             |
| `rm -d <dir>`        | Удалить папку                            |
| `asciiart`           | Показать ASCII-арт логотипа              |
| `cpuid` / `memtest`  | Информация о процессоре / тест памяти    |
| `rand`               | Случайное число                          |
| `date -d/-t`         | Дата и время                             |
| `pause`              | Пауза до нажатия клавиши                 |

## 🔧 Сборка и запуск

### Зависимости
- `gcc-multilib`, `nasm`, `ld`, `xorriso`, `grub-mkrescue` (для ISO)
- `qemu-system-i386` (для тестирования)

### Сборка
```bash
git clone https://github.com/yourname/yodaos.git
cd yodaos
make disk.img      # создать пустой образ диска 64 МБ
make               # собрать ISO
make run           # запустить в QEMU
